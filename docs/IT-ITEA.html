Description

Generic implementation of Interaction-Transformation Evolutionary Algorithm for any instance of IT expression.

To run itea you just need to call 'itea mutFun pop0', where mutFun is a mutation function of the type 'Mutation a b', and pop0 is the initial 'Population a b' of solutions. This function will result in an infinite list of populations, with the i-th element being the population of the i-th generation.

This library also provides some generic mutation function builders.
